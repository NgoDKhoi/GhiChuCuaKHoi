# Ghi Chú Phím Tắt

| Tổ hợp phím         | Chức năng                         |
|---------------------|-----------------------------------|
| **Ctrl + D**        | Thêm trang hiện tại vào bookmark  |
| **Ctrl + Shift + X**| Mở extension trên VS Code            |
| **Ctrl + Shift + V**| Mở Preview bookmark         |
| **Ctrl + T**        | Mở tab mới                        |
| **Ctrl + W**        | Đóng tab hiện tại                 |
